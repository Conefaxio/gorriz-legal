Privacy Policy

Gorriz IA Code Assistant does not collect, store, transmit, or process personal data.

The plugin embeds the official Perplexity.ai web interface using JetBrains JCEF technology.

All authentication, conversations, and data processing occur directly between the user and Perplexity.ai.

The plugin does not:

  • Access user credentials;
  • Store authentication tokens;
  • Intercept messages;
  • Track usage or analytics;
  • Transmit data to external servers.

The author has no access to user conversations or data.
